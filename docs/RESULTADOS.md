## Estados

| Estado | Significado | Acción |
|--------|-------------|--------|
| 🔴 **VULNERABLE** | Sin mitigación | Aplicar mitigación AHORA |
| 🟠 **PROBABLEMENTE VULNERABLE** | No se pudo verificar | Habilitar SSM/SSH |
| 🟡 **MITIGADO** | Mitigación aplicada | Esperar parche |
| 🟢 **NO VULNERABLE** | Kernel parcheado | Ninguna |
| ⚪ **DESCONOCIDO** | No se pudo conectar | Verificar conectividad |
